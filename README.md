## Objective
Implement and analyze core graph algorithms including BFS, DFS, Topological Sort, Shortest Path (Dijkstra/Bellman-Ford), and Minimum Spanning Trees (Kruskal) to solve practical, real-world networking and routing problems.

## Setup & Usage
1. `python -m venv .venv`
2. Activate environment.
3. `pip install -r requirements.txt`
4. Run `jupyter lab` and open `notebooks/graph_realworld.ipynb`

## Problem Summary
* **Traversal:** BFS/DFS applied to friend suggestion networks.
* **Topological Sort:** Task dependency management.
* **Shortest Path:** Dijkstra for emergency routing; Bellman-Ford for routes with negative weights.
* **MST:** Kruskal's algorithm for cost-efficient cabling.
